## 📑 프로젝트 설명 
- MySQL JDBC 환경으로 구축된 프로그램을 클론 코딩 후 MyBatis를 사용하도록 코드를 수정하였습니다.
- MyBatis를 사용하면서 MVC2 패턴을 적용하는 코드로 수정하고 있습니다.
  
### 구현 기능
1. 게시글 추가
2. 게시글 조회/수정/삭제
3. 게시글 전체 삭제

### 향후 작업 방향
✅ MyBatis로 JDBC 수정    
⏩ MVC2 모델을 적용하여 수정 -> ~ing
